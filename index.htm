Remove Source Code from the Spotlight results

lion 
defaults write com.apple.spotlight orderedItems -array-add '{ "enabled" = "0" ; "name" = "SOURCE"; }'

Open screen sharing

leopard 
open vnc://computername.local

Show hidden files

leopard 
defaults write com.apple.finder AppleShowAllFiles true

2D Dock

lion leopard 
defaults write com.apple.dock no-glass -boolean YES 
killall Dock

Make hidden Applications Icons semi-transparent

lion leopard 
defaults write com.apple.Dock showhidden -bool YES 
killall Dock

Re-Index Spotlight

lion leopard 
sudo mdutil -E -p "/Volumes/Macintosh HD"

Screencapture

lion leopard

Screenshot file format

defaults write com.apple.screencapture type png

Screenshot name

defaults write com.apple.screencapture name ScreenShot

Screenshot Location

defaults write com.apple.screencapture location [path]

Disable Time Machine warning

lion leopard 
defaults write com.apple.TimeMachine DoNotOfferNewDisksForBackup -bool YES

Disable .ds-store on network shares

lion leopard 
defaults write com.apple.desktopservices DSDontWriteNetworkStores true

Uninstalling Xcode Developer Tools

lion leopard 
sudo /Developer/Library/uninstall-devtools --mode=all
